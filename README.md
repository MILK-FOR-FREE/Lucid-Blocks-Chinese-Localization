# Lucid Blocks汉化/简体中文/繁體中文本地化补丁

本补丁为 **Lucid Blocks** 的 简体中文/繁體中文 本地化文件，旨在为玩家提供完整的中文游戏体验，会随着时间不断改进与游戏同时走向完美。

**繁體中文以台灣繁體為準。**

---

##  常规安装方法
1. **下载补丁**：
   前往右侧的 [Releases] 页面，下载对应版本的压缩包.
   
   若是简体中文，下载对应版本的 `zhswdkls` ，若是繁體中文，下载对应版本的  `zjc`。
   
   * *注：若暂无对应游戏版本的压缩包，请优先下载最新的版本，更新将随后发布。*
2. **定位目录**：
   在 Steam 库中右键点击 `Lucid Blocks` -> `管理` -> `浏览本地文件`。
3. **置入文件**：
   解压下载好的压缩包，将里面的 `mods` 文件夹整体拖入游戏根目录中。
   * **正确路径结构应如下所示：**
   ```text
   lucid-blocks/                <-- 游戏根目录
   └── mods/                    <-- mods文件夹
       └── .pck                 <-- 汉化文件
4. **开始游戏**：
   直接通过 Steam 启动游戏，汉化将自动生效。
##  从旧版本升级至3.0.4版本后的安装方法
**作者在这个版本后修改了目录的文件顺序，只需要手动把mods文件夹移动即可**
1. **继续定位目录**：
   在 Steam 库中右键点击 `Lucid Blocks` -> `管理` -> `浏览本地文件`。
2. 找到其中的 `lucid-blocks` 文件夹
   进入此文件夹，找到原先的 `mods` 文件夹
   
   将 `mods` 文件夹手动移动至上一层 `lucid-blocks` 文件夹
   * **正确路径结构应如下所示：**
   ```text
   lucid-blocks/                <-- 游戏根目录（新文件夹）
   └── mods/                    <-- mods文件夹
       └── .pck                 <-- 汉化文件
   lucid-blocks                 <-- 旧文件夹
4. **开始游戏**：
   删除旧的 `lucid-blocks` 文件夹，直接通过 Steam 启动游戏，汉化将自动生效。

## 补丁状态说明
游戏已被100%汉化并逐渐趋向完美且被官方认可，并参与其他社区内容的制作。但由于游戏文本量较大且游戏环境复杂，补丁可能存在以下情况：
* **风格声明**：**原游戏的原文风格本身就非常神叨叨且夸张，汉化补丁中的内容不是我故意为了私人原因瞎写的，是它原文就这样，我只做到了更还原的中文表达！！！游戏发布前就在准备，又是看古文又是查宗教神话，又得看原文还得结合中文，也听取实际游戏体验和群友一起改进。所以不要因为游戏翻译超出常规就说我ai或者机翻拜托了**
* **特殊名词**：部分文本包含原作者的自创词汇，通过参考语言学与宗教神话故事，可通过拆解梵语、拉丁语、希腊语和阿拉伯语等古语言词根进行意译。个别内容仍在持续推敲，以求找到更契合的表达。
* ~~**语义偏移**：部分文本缺乏明确语境，可能存在翻译偏差或一词多义处理不当的情况。~~ （目前已不存在这样的问题）
* ~~**翻译遗漏**：部分特定物品或底层文本可能存在漏译.~~ （目前已不存在这样的问题）
* 以上都将随着时间逐渐完美。
